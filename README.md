# Alaya Pulse

A premium live polling web app for Alaya internal use — Mentimeter alternative without audience caps.

**Status:** In active development. Not yet deployed.

## What it is

- Audience joins on phones via QR code, 6-character code, or short URL
- Presenters build decks mixing imported PDF slides and native question slides
- Question types: MCQ, Word Cloud, Open-ended, Rating (more in Phase 2)
- Live vote sync via Firestore
- Free to use; designed to support 600+ concurrent audience members per session

## Tech stack

React + TypeScript + Vite · Tailwind CSS · shadcn/ui · Framer Motion · Magic UI · Aceternity UI · Firestore · pdf.js · Vercel hosting

## Getting started

See [`HANDOFF.md`](./HANDOFF.md) for full project context, decisions, and next steps.
